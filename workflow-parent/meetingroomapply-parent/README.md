# meetingapply-parent

##### 会议室使用申请工作流
  * api：后台的service和dao的接口
  * biz：后台service接口的实现
  * dao：后台dao接口的实现
  * entities：javaBean实体类
  * extjs：前台增删改查界面的展示
  * rest：前台到后台的rest服务配置以及工作流定义文件
  * web：前台菜单的实现

# 流程说明
  会议使用申请开始->校务部文秘综合干事审批->校务部行政事务办主管审批->校务部副部长审批->发起部门会议纪要审批->结束

# 注意事项
  * 会议室使用申请时需要控制该会议室是否被占用
